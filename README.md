# Dayflow â€“ Human Resource Management System

Dayflow is a simple Human Resource Management System (HRMS) developed as part of the **Odoo Hackathon**.  
This project focuses on digitizing basic HR operations such as employee authentication, attendance tracking, and leave management.

The solution is built **from scratch using core technologies** without using any third-party frameworks or APIs.

---

## Problem Statement
The problem statement was provided by **Odoo**, where the objective was to design a basic and efficient HR system that can be integrated into an ERP-like environment.

---

## Features
- User Authentication (Login / Signup)
- Role-based access (Admin / Employee)
- Employee profile management
- Attendance tracking
- Leave request and approval workflow
- Simple and clean UI

---

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL

---

## Project Approach
- Developed as a **solo project**
- Focused on completing the **core workflow end-to-end**
- No external libraries, frameworks, or third-party APIs were used
- Designed keeping simplicity and clarity in mind

---

- Payroll automation
- Performance reports
- Improved UI/UX
- Integration with ERP modules
